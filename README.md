#  NYC Green Taxi – Projet Data Engineering sur Azure

##  Présentation

Ce projet illustre une pipeline simple de Data Engineering sur Microsoft Azure à partir du dataset NYC Green Taxi.

Les données sont stockées dans Azure Data Lake (couche Gold), transformées avec Databricks, puis analysées via Azure Synapse Analytics et Power BI.

---

##  Architecture simplifiée

Dataset NYC Taxi  
→ Azure Data Lake (container Gold)  
→ Transformation avec Azure Databricks  
→ Requêtes analytiques avec Azure Synapse  
→ exposition dans Power BI  

---

##  Technologies utilisées

- Azure Data Lake Storage Gen2  
- Azure Databricks  
- Azure Synapse Analytics  
- Power BI  

---



---

##  Objectif du projet

Mettre en place une architecture simple de type Lakehouse sur Azure et démontrer les compétences suivantes :

- Manipulation de données dans le cloud  
- Transformation avec Spark  
- Requêtes SQL analytiques  
- Exposition des données pour la BI
